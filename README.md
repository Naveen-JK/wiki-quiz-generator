# Wiki Quiz Generator

## Overview
The Wiki Quiz Generator is an application that allows users to create and take quizzes based on Wikipedia articles. It consists of a backend service that handles quiz data and a frontend interface for users to interact with.

## Features
- Create quizzes from Wikipedia articles
- Take quizzes and receive instant feedback
- User-friendly interface

## Project Structure
- **backend/**: Contains the backend source code and services.
- **frontend/**: Contains the frontend source code and components.

## Getting Started
To get started with the project, clone the repository and install the necessary dependencies for both the backend and frontend.

### Backend
1. Navigate to the `backend` directory.
2. Install dependencies using `npm install`.
3. Start the server with `npm start`.

### Frontend
1. Navigate to the `frontend` directory.
2. Install dependencies using `npm install`.
3. Start the frontend application with `npm start`.
