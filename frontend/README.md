# README for Frontend Wiki Quiz Generator

This is the frontend part of the Wiki Quiz Generator application. It is built using React and TypeScript.

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd wiki-quiz-generator/frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Running the Application

To start the development server, run:
```bash
npm start
```

The application will be available at `http://localhost:3000`.

## Folder Structure

- `src/components`: Contains React components for the application.
- `src/services`: Contains API service functions to interact with the backend.
- `src/types`: TypeScript type definitions used throughout the application.
